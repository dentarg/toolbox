# toolbox.elixir.pm

WIP: A site to list Hex packages by category. Inspired by <http://ruby-toolbox.com>.


## Development

    mix deps.get
    mix ecto.create && mix ecto.migrate
    mix phoenix.server

Now visit <http://localhost:4000>.


## Test

    mix test


## Production

Deployed to Dokku.

    git push dokku
